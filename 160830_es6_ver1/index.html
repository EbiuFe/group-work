es6の注意点

・thisがclassを見るようになるのは、アロー関数使用時のみみたいなので、一緒にprototypeの記述を混ぜるとthisがおかしくなる

※アロー関数内部のthisは関数定義時に決定します

従来のjsのthisは「関数がどのようにして呼ばれたか」によって変化する。

アロー関数の書き方

１）var fn = () => {console.log('hoge');};引数がない場合、()は省略不可

２）var fn = (x) => {console.log(x);};引数がが一つの場合

３）var fn = x => {console.log(x);};引数がが一つの場合、()を省略可能

４）var fn = (x, y, z) => {console.log(x, y, z);};複数の場合は、カンマで区切る

５）(() => { console.log(); })();即時関数

６）clickイベントなども書き方注意

jsの場合は
(function () { /* 関数本体 */ }());
(function () { /* 関数本体 */ })();
どちらでも実行されるが、アロー関数は後者のみでないとエラーになるので注意。
